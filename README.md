# RetrofitLab - Unidad 5

Proyecto desarrollado para el Post-Contenido 1 de la Unidad 5 de Aplicaciones Móviles.

## Descripción

Aplicación Android desarrollada en Kotlin con Jetpack Compose que consume la API pública JSONPlaceholder mediante Retrofit y OkHttp. La aplicación muestra una lista de posts, permite cargar más elementos mediante paginación simulada y maneja estados de interfaz como carga, éxito y error de red.

## Tecnologías utilizadas

- Kotlin
- Jetpack Compose
- Retrofit
- OkHttp
- HttpLoggingInterceptor
- kotlinx.serialization
- Coroutines
- StateFlow
- ViewModel
